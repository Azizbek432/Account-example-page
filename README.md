# IT-Park: HTML & CSS Amaliy Mashg'ulot - Profile Card

Ushbu amaliy mashg'ulot **IT-Park** darslari davomida olingan bilimlarni mustahkamlash uchun yaratildi. Asosiy e'tibor CSS-da elementlarni vizual jihatdan chiroyli ko'rinishga keltirish va ularni konteyner ichida to'g'ri joylashtirishga qaratilgan.

## 🛠 Ishlatilgan texnologiyalar
* **HTML5** — Kartochka strukturasi (rasm, sarlavha, matn va tugmalar).
* **CSS3** — Dizayn, ranglar va joylashuv (layout) nazorati.

## 📝 Loyiha xususiyatlari
Ushbu mashg'ulotda quyidagi CSS konseptlari muvaffaqiyatli qo'llanildi:

* **Centering:** Butun kartochkani sahifaning o'rtasiga va kartochka ichidagi elementlarni markazga tekislash.
* **Box Model:** `padding`, `margin` va `border` yordamida elementlar orasidagi masofalarni sozlash.
* **Border Radius:** Profil rasmini aylana shakliga keltirish va kartochka burchaklarini yumshatish.
* **Buttons & Badges:** "Skills" qismi uchun alohida badge-lar va interaktiv "Follow/Message" tugmalari.
* **Typography:** Shriftlar ierarxiyasi (H2, H3, P) va ranglar kontrastini to'g'ri tanlash.

# IT-Park: HTML & CSS Practice - Profile Card

This practice project was created to reinforce the skills learned during **IT-Park** lessons. The main focus was on styling elements to create a clean UI and mastering positioning within a container.

## 🛠 Technologies Used
* **HTML5** — Card structure (image, headings, text, and buttons).
* **CSS3** — Design, colors, and layout control.

## 📝 Project Features
The following CSS concepts were successfully implemented in this practice:

* **Centering:** Aligning the entire card and its inner content to the center.
* **Box Model:** Managing spacing using `padding`, `margin`, and `border`.
* **Border Radius:** Creating a circular profile image and rounding card corners.
* **Buttons & Badges:** Designing custom skill badges and interactive "Follow/Message" buttons.
* **Typography:** Proper use of font hierarchy (H2, H3, P) and color contrast.
